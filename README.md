# 🔥 Ignite Academy - Advanced Learning Management System

Ignite Academy is a premium, full-stack Learning Management System (LMS) designed to provide a seamless educational experience for both students and instructors. Built with a modern tech stack, it features secure payments, real-time progress tracking, and a powerful admin dashboard.

## Key Features

### For Students
- **Course Marketplace:** Explore a wide range of courses with detailed descriptions and previews.
- **Secure Payments:** Seamless checkout experience powered by **Stripe**.
- **Interactive Course Player:** Track your lessons with a sleek, distraction-free video player.
- **Progress Tracking:** Real-time completion status for every course and lesson.
- **Personalized Dashboard:** Overview of enrolled courses, stats, and learning journey.

###  For Admins
- **Course Builder:** Robust CRUD operations for courses, modules, and lessons.
- **Sales Analytics:** Track revenue and enrollment stats with intuitive charts.
- **User Management:** Manage student accounts and roles with ease.
- **Dashboard Overview:** High-level metrics on platform performance.

###  Core Systems
- **Authentication:** Secure JWT-based auth with HTTP-only cookies.
- **Email Services:** Automated verification and password reset via **Nodemailer**.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using **Tailwind CSS 4**.

## 🛠️ Technology Stack

### Frontend
- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS 4
- **Routing:** React Router 7
- **Icons:** Lucide React
- **State/Data Fetching:** Axios

### Backend
- **Server:** Node.js & Express 5
- **Database:** PostgreSQL (Cloud or Local)
- **ORM:** Prisma 7
- **Authentication:** JSON Web Tokens (JWT) & BcryptJS
- **Payments:** Stripe API
- **Emails:** Nodemailer
